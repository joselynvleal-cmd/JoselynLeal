Instrucciones de Navegación
1. Descarga o clona el repositorio en tu computadora.
2. Abre la carpeta del proyecto en Visual Studio Code
3. Abre el archivo principal index.html en el navegador web (se recomienda Google Chrome, Microsoft Edge o Mozilla Firefox). 
   Alternativa: Puedes usar la extensión Live Server en Visual Studio Code haciendo clic derecho sobre index.html y seleccionando "Open with Live Server"


Descripción de las Secciones

Inicio de Sesión (Login): Pantalla de acceso centrada con validación de correo electrónico y contraseña (mínimo 8 caracteres), opción de "Recordarme" y enlaces de recuperación

Registro de Clientes: Formulario estructurado en un diseño de dos columnas que incluye datos personales  (nombre, cédula y teléfono con restricciones numéricas exactas), selector de tipo de cliente, sector de industria y campo de dirección

 Tecnologías Utilizadas

HTML5: Estructuración semántica utilizando etiquetas como `<header>`, `<nav>`, `<main>`, `<section>`, `<form>`, `<fieldset>`, `<legend>` y `<label>`
CSS3: Estilos personalizados aplicando el Modelo de Caja (`box-sizing: border-box`), **Flexbox** para centrado y alineación, **CSS Grid** para la distribución de formularios, variables CSS (`:root`) para una paleta de colores coherente, y selectores avanzados con transiciones suaves (`:hover`, `:focus`, `:valid`, `:invalid`)
* Diseño Responsive: La página se adapta sin problemas a cualquier pantalla, ya sea un teléfono, una tablet o la computadora, usando media queries para que el contenido siempre se vea ordenado y cómodo de usar.

Para probar el diseño responsive, puedes hacerlo de la siguiente manera:
 Abre la página en tu navegador, presiona la tecla **F12**, haz clic en el icono de un teléfono/tablet (en la esquina superior izquierda del panel que se abre) y selecciona cualquier modelo de celular para ver cómo se adapta al instante.
